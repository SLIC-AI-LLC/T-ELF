**TELF.factorization.SymNMFk:** Symmetric Non-negative Matrix Factorization with Automatic Model Determination
-----------------------------------------------------------------------------------------------------------------------

SymNMFk is a Symmetric Non-negative Matrix Factorization module with the capability to do automatic model determination.

See example usage `here <https://github.com/lanl/T-ELF/tree/main/examples/SymNMFk>`_
